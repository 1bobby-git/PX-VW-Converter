VW 계산기 · VW 변환기 (PX ↔ VW)

VW 계산기와 VW 변환기로 픽셀(PX)과 뷰포트 너비(VW) 단위를 쉽고 정확하게 변환하세요.

계산 기준이 되는 화면 너비입니다.

표시할 소수점 자릿수를 선택하세요.

PX → VW 변환

픽셀 값을 뷰포트 너비 단위로 변환합니다.

VW → PX 변환

뷰포트 너비 단위를 픽셀 값으로 변환합니다.

CSS 소스 변환

CSS 코드의 PX 값을 VW로 일괄 변환합니다.

사용법 및 팁

PX → VW 변환

  • PX 값을 입력하면 자동으로 VW로 변환됩니다
  • 뷰포트 너비를 기준으로 계산됩니다
  • 소수점 자릿수를 조정할 수 있습니다

VW → PX 변환

  • VW 값을 입력하면 자동으로 PX로 변환됩니다
  • 지정된 뷰포트 너비를 기준으로 계산됩니다
  • 반응형 디자인에 유용합니다

PX → VW: VW = (PX ÷ 뷰포트너비) × 100

VW → PX: PX = (VW ÷ 100) × 뷰포트너비
